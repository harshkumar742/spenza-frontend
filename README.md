# Spenza Webhook Frontend

## Tech Stack
- React.js
- Axios
- JWT Auth
- React Router

## Setup

### 1. Clone the repo
```bash
git clone https://github.com/harshkumar742/spenza-frontend
cd spenza-frontend
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run
```bash
npm start
```

### 4. Environment
This application runs on http://localhost:3000.
API is assumed to run on `http://localhost:8000`.
