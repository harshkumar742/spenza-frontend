# Spenza Webhook Frontend

## Tech Stack
- React.js (Vite or CRA)
- Axios
- JWT Auth
- React Router

## Setup

### 1. Clone the repo
```bash
git clone https://github.com/your-repo/spenza-frontend.git
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run
```bash
npm start
```

### 4. Environment
API is assumed to run on `http://localhost:8000`.
